
# TBA

## Overview 
Python client API (v5.2) for [RavenDB](https://ravendb.net/) , a NoSQL document database.

## Installation
Install from [PyPi](https://pypi.python.org/pypi), as [ravendb](https://pypi.python.org/pypi/ravendb).
```bash
pip install ravendb
````
