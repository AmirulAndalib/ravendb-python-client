TBA
====

Overview
===============

Python client API for `RavenDB <https://ravendb.net>`_, a NoSQL document database. v5.2

Installation
=============
Install from `PyPi <https://pypi.python.org/pypi>`_, as `ravendb <https://pypi.python.org/pypi/ravendb>`_.

.. code-block::

  pip install ravendb
